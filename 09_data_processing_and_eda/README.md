# Data Processing and EDA

Goal: learn how to inspect data before training a model.

Open the notebook:

```text
09_data_processing_and_eda/eda_titanic.ipynb
```

Run each cell from top to bottom.

This notebook shows:

- loading a CSV
- viewing rows
- checking shape
- checking missing values
- summarizing numeric columns
- checking target balance
- grouping by features
- making simple plots
- creating a new feature

Use notebooks for exploration. Use `.py` files for clean training scripts.
